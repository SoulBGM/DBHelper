《首页 测试接口:

  • 查看所有的年级信息
  • 查询年级信息
  • 添加年级信息
  • 获取配置文件数据

查看所有的年级信息

method：GET
url: /demo/queryAllGrade

参数列表

参数名类型必需描述

返回结果: Android Code | IOS Code

{
  "gradeId": "long //ID标识",
  "gradeName": "string //年级名称"
}

查询年级信息

method：GET
url: /demo/queryById/{id}

参数列表

参数名类型必需描述
idlongfalseID标识

返回结果: Android Code | IOS Code

{
  "gradeId": "long //ID标识",
  "gradeName": "string //年级名称"
}

添加年级信息

method：POST
url: /demo/addGrade

参数列表

{
  "gradeId": "long //ID标识",
  "gradeName": "string //年级名称"
}

返回结果: Android Code | IOS Code

{
  "success": "boolean //是否成功",
  "msg": "string //消息",
  "obj": {}
}
